Swordinee

This is the game which I have been putting (albeit not enough) of my free time into. This is a passion project of mine, where all the assets and code were created by me. I do have a path in which I would like to take this project moving forward, though it may not be explicitly clear at the moment. The build at the current moment is a very early prototype version with bare-bones features and a single level, but I felt it was important for me to put a project in my portfolio that isn't from a class. I don't believe the current state of the game is the height of my game development ability, but I hope it gets the point across that I am capable of creating the whole of a cohesive game in a short(ish) amount of time. More updates coming soon.
